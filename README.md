Phundament
==========

Version 4.0.1.0-dev

** DO NOT USE FOR PRODUCTION YET! **

Vanilla Yii2 Application with composer support
----------------------------------------------

### Get started

Install application    

    composer.phar create-project -sdev phundament/app-vanilla2 

Setup application

    chmod 777 ./app-vanilla2/web/assets

Open application

    http://localhost/app-vanilla2/web
